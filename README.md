# CHAT-APPLICATION

## PROJECT OVERVIEW

This project is a real-time chat application developed using React.js, Node.js, and Socket.IO. It enables users to send and receive messages instantly through WebSocket communication.

## FEATURES

- Real-Time Messaging
- Frontend and Backend Integration
- Socket.IO Communication
- Responsive Interface
- Live Message Updates

## TECHNOLOGIES USED

- React.js
- Node.js
- Express.js
- Socket.IO
- CSS3

## PROJECT STRUCTURE

CHAT-APPLICATION/

client/
server/
README.md

## HOW TO RUN

### Server

cd server

npm install

npm start

### Client

cd client

npm install

npm start

The application runs on:

http://localhost:3000

## CONCLUSION

The project demonstrates full-stack real-time communication using Socket.IO with seamless frontend and backend integration.
